## About

<p>
    Проект для списка сотрудников в компании, с функционалом добавления, редактирования, удаления сотрудников, авторизацией/регистрацией, приватным роутингом.
</p>

## Project setup

```
npm install
npm run start
npm run test
npm run lint:fix
npm run lint:scss:fix
```
